# Mini Muyu 🎙️

Mini Muyu is a modern mobile application developed with **React Native** and **Expo**, designed to capture and organize audio recordings.
## 🛠️ Tech Stack

- [Expo](https://expo.dev/)
- [React Native](https://reactnative.dev/)
- [NativeWind](https://www.nativewind.dev/) (Tailwind CSS)
- [React Navigation](https://reactnavigation.org/)
- [Supabase](https://supabase.com/)
- [Lucide Icons](https://lucide.dev/)
- [Expo AV](https://docs.expo.dev/versions/latest/sdk/av/) (Audio & Video)

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:CristhianOcola/Mini-muyu.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure your environment variables in `src/lib/supabase.js` (Supabase URL and Anon Key).

4. Start the project:
   ```bash
   npx expo start
   ```
